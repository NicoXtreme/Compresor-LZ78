# Compresor LZ78

Aplicación con interfaz gráfica para comprimir y descomprimir archivos de texto utilizando el algoritmo LZ78.

## Características

- Compresión y descompresión con algoritmo LZ78
- Interfaz gráfica intuitiva
- Visualización del diccionario generado
- Estadísticas de rendimiento
- Validación de archivos
- Soporte para formato personalizado .lz78

## Requisitos

- Python 3.8+
- Tkinter (incluido en Python)
